# Как работают ссылки в боте

## Никакого REST — всё внутри Телеграма

Ссылки реализованы через **встроенный механизм Telegram Deep Linking** (глубокие ссылки). Никакого веб-сервера, REST API или внешних сайтов не используется.

## Как это устроено

### 1. Генерация ссылки

Когда пользователь пишет `/start`, бот:
1. Вызывает `get_or_create_link(user.id)` — создаёт в БД запись `ChatLink` с уникальным кодом
2. Код генерируется через `secrets.token_urlsafe(16)` — выглядит как `abc123_def-ghi`
3. Склеивает ссылку: `https://t.me/{username_бота}?start={код}`

```python
link = await get_or_create_link(user.id)
share_url = f"https://t.me/{get_bot_username()}?start={link.code}"
# → https://t.me/MyAnonBot?start=abc123_def-ghi
```

### 2. Открытие ссылки

Когда человек нажимает на эту ссылку на телефоне:
- **Android / Desktop:** Telegram открывается, показывает кнопку «НАЧАТЬ» или сразу отправляет команду
- **iPhone (iOS):** iOS сначала открывает Safari, который перекидывает в Telegram через `tg://` протокол

Telegram отправляет боту сообщение: `/start abc123_def-ghi`

### 3. Обработка на боте

`start_handler` ловит команду через `CommandStart()`:
```python
@dp.message(CommandStart())
async def start_handler(message: Message, command: CommandStart):
    code = command.args  # "abc123_def-ghi"
    if code:
        # Ищем ссылку по коду в БД
        link = await get_link_by_code(code)
        # Создаём активную сессию
        await set_active_session(message.from_user.id, code)
        # Готово — теперь пользователь может писать анонимно
```

### 4. Почему может не работать на iPhone

Есть 3 типичные проблемы:

| Проблема | Почему | Решение |
|---|---|---|
| Ссылка не кликабельна | Текст ссылки разорван переносом строки или внутри `<code>` тега | Убедиться, что URL на отдельной строке без тегов |
| Safari вместо Telegram | iOS не умеет сразу открывать `t.me` — сначала открывает Safari | Это нормально, Safari должен перекинуть в Telegram. Если не перекидывает — надо обновить iOS |
| Ссылка "пропадает" | HTML-разметка битая (например, `<b>` без `</b>`) | Проверить что все HTML-теги закрыты |

Сейчас ссылка формируется так:
```
📌 <b>Твоя ссылка для анонимных сообщений:</b>
https://t.me/MyAnonBot?start=abc123_def-ghi
```
— URL на отдельной строке, вне HTML-тегов. Это корректный формат.

### Если на iOS не открывается

Проверь:
1. Есть ли на телефоне установлен Telegram
2. Достаточно ли новая версия iOS/Telegram
3. Не включена ли блокировка переходов по внешним ссылкам в настройках Telegram

Это не баг бота — это особенности обработки `t.me` ссылок на iOS.
