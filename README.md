# 🤖 Анонимный Telegram Bot

Бот для анонимного общения. Каждый пользователь получает уникальную ссылку. Любой, кто перейдёт по ссылке, может написать владельцу анонимно. Владелец может ответить — и ответ тоже уйдёт анонимно.

Двусторонняя анонимная переписка.

---

## 🚀 Быстрый старт

```bash
git clone https://github.com/JamikKhidirov/AnonimBot.git
cd AnonimBot
pip install -r requirements.txt
```

Создай `.env` (скопируй `.env.example`):
```
BOT_TOKEN=
DEVELOPER_ID=
```

Запуск:
```bash
python -m bot
```

## 🎛 Админ-панель (интерактивная)

После `/admin` открывается меню с кнопками:

```
[📊 Статистика]  [👥 Пользователи]
[✉️ Сообщения]   [👑 Админы]
[🔍 Поиск]       [🛠 Инструменты]
[⚙️ Для разработчика]
```

### Разделы

**📊 Статистика** — количество пользователей, ссылок, сообщений

**👥 Пользователи** — список всех пользователей с ролями

**✉️ Сообщения** — постраничный список (по 5), можно нажать на номер и увидеть:
- кто отправитель (ID, username, имя)
- сколько всего сообщений от этого человека
- кнопки «Все сообщения от отправителя» и «Профиль»

**👑 Админы** — список, добавить/удалить админа

**🔍 Поиск** — поиск пользователя или сообщения по тексту, просмотр по ID

**🛠 Инструменты:**
- ⛔ Бан / Разбан
- 📋 Список забаненных
- 📢 Рассылка всем пользователям
- 🧹 Очистка старых сообщений
- 📤 Экспорт CSV
- 🔄 Сброс ссылки

## 🔗 Как работают ссылки

Ссылка выглядит так: `https://t.me/MyBot?start=abc123...`

При нажатии Telegram отправляет боту `/start abc123...`. Бот находит код в БД и создаёт сессию — никакого REST, всё внутри Telegram.

Подробнее — в [HOW_LINKS_WORK.md](HOW_LINKS_WORK.md).

## 💬 Двусторонняя переписка

1. Аноним отправляет сообщение → владельцу приходит уведомление
2. Владелец отвечает реплаем → ответ уходит анониму
3. Аноним отвечает реплаем → ответ уходит владельцу
4. И так далее — полноценный диалог

Если владелец админ/разработчик — к сообщению прикрепляется кнопка «👤 Кто это?», показывающая данные отправителя.

## 🗄 База данных

SQLite через SQLAlchemy (async).

### Таблицы

| Таблица | Назначение |
|---------|-----------|
| `users` | Пользователи (ID, username, имя, роль, язык) |
| `chat_links` | Ссылки для анонимного общения |
| `messages` | Анонимные сообщения |
| `active_sessions` | Активные сессии анонимов |
| `forwarded_messages` | Привязка сообщений бота к оригинальным (для реплаев) |
| `admins` | Аудит лог назначений админов |
| `banned_users` | Забаненные пользователи |

## 📝 Логирование

В консоль пишутся все события:
```
ANON: sender=123456 (@user) -> owner=789012 (@owner): текст...
REPLY (owner->sender): 789012 -> 123456: ответ...
BROADCAST: отправитель=789012, всего=150, отправлено=148, ошибок=2
```
